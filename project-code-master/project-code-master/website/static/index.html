Instructions:

Controls

The micro:bit uses the left and right buttons and tilt controls as user input. As labelled on the device, the left button will be referred to as A and the right button as B.
To return to the start of the login process at any time, the small button on the back of the device can be pressed.

Menu navigation

B - view next app
A - view previous app
Tilt away - select current app
Tilt towards - return to previous menu

Keyboard navigation

B - view next letter
A - view previous letter
B while tilted right - view 5 letters forward
A while tilted left - view 5 letters back
Tilt away - select current letter
Tilt towards - finish keyboard input

Lock screen

The micro:bit lock screen uses a pattern lock, similar to many smartphones. You can set your pattern on this website.

When logging in for the first time on a micro:bit, you will be prompted for your username. After entering your username, you will be prompted for your lock pattern.

The display on the micro:bit shows the cursor in the centre of the screen, and each of the dots as a 3x3 box. Tilting the micro:bit moves the cursor around the grid- the cursor will remain in the middle of the screen while the section of the grid being viewed changes. Holding the micro:bit horizontally will keep the cursor in the same position.

           
Cursor hovering over the bottom left dot. Bottom left dot selected. Selecting bottom left, bottom middle, then middle left dots.

To log in, use the cursor and either the A or B button to select the dots in the correct pattern.

Hold the A and B buttons simutaneously after entering the correct pattern in order to log in.
If an incorrect pattern is entered, you will be taken back to the start of the login process.

User Interface

The micro:bit interface is formed of apps, with a symbol used to represent each app.
Top Level: Messaging (M), micro:bit ID (I), Weather (W)
Messaging: Send message (↑), Receive message (↓)
